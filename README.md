# Konexão Direta (KD)

Este repositório contém a implementação inicial da plataforma **Konexão Direta (KD)**, um marketplace SaaS B2B2P de governança e intermediação.

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute o aplicativo Streamlit:
   ```bash
   streamlit run streamlit_app.py
   ```

Consulte o arquivo `SPEC.md` para detalhes completos sobre as regras de negócio e arquitetura.
